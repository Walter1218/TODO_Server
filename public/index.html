💡 快速入门指南

1. 首先创建一个智能体（Agent）

2. 为该智能体添加 TODO 任务

3. 通过 API 管理任务生命周期

📡 API 端点

👤 智能体管理

POST /api/agents
注册新智能体
GET /api/agents
获取所有智能体
GET /api/agents/:id
获取特定智能体

✅ TODO 任务管理

POST /api/agents/:agentId/todos
创建新任务
GET /api/agents/:agentId/todos
获取所有任务（支持筛选）
GET /api/agents/:agentId/todos/stats
获取统计信息
GET /api/agents/:agentId/todos/search?q=keyword
搜索任务
PUT /api/agents/:agentId/todos/:id
更新任务
PATCH /api/agents/:agentId/todos/:id/status
更新任务状态
DELETE /api/agents/:agentId/todos/:id
删除任务

💻 使用示例

创建智能体

# cURL 示例
curl -X POST http://localhost:3000/api/agents \
  -H "Content-Type: application/json" \
  -d '{
    "name": "claude-agent",
    "metadata": {
      "model": "claude-3",
      "version": "1.0"
    }
  }'

创建任务

# cURL 示例
curl -X POST http://localhost:3000/api/agents/{agent_id}/todos \
  -H "Content-Type: application/json" \
  -d '{
    "title": "分析用户需求",
    "description": "分析并整理用户反馈",
    "priority": "high",
    "context": "这是Q2季度的核心功能",
    "tags": ["需求分析", "核心功能"]
  }'

查询任务（按优先级筛选）

# 获取所有高优先级任务
curl "http://localhost:3000/api/agents/{agent_id}/todos?priority=high"

🎯 核心特性

🔢 优先级系统
支持 4 个优先级：critical（紧急）、high（高）、medium（中）、low（低）
🏷️ 标签管理
为任务添加多个标签，方便分类和筛选
📝 上下文追踪
记录任务背景和来源，减少长距离复杂任务中的遗忘
🔍 全文搜索
支持在标题、描述、上下文中搜索关键词
📊 统计报表
实时统计任务状态分布，监控任务进度
